# Disc Golf Tag Masters

A mobile focused web application to help run disc golf tag matches

## Technologies Used

- Node.js
- Express
- MongoDB
- Mongoose
- jQuery
- Javascript
- Embedded Javscript (ejs)
- HTML5
- CSS3
